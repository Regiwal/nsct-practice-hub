# nsct-practice-hub
A modern web app for practicing National Skill Competency Test (NSCT) MCQs with category-based quizzes, results, and answer review system.


A modern and responsive web application designed to help students prepare for the National Skill Competency Test (NSCT) through structured MCQ practice.

---

## 🔥 Features

- 📚 Category-based MCQ system (Web, DB, Programming, etc.)
- 🧠 Practice Mode (instant learning)
- ⏱️ Mock Test Mode (real exam simulation)
- 📊 Result calculation with score & percentage
- ✅ Answer review with correct/incorrect highlights
- 🔎 Easy to scale (500+ MCQs supported)
- 📱 Fully responsive design

## 📌 How It Works

- MCQs are stored in separate JSON files by category
- Questions are dynamically loaded using JavaScript (fetch API)
- Users can attempt quizzes and view results instantly
